Week 1

Zoek minimaal 10 (JavaScript) libraries en beschrijf in (ongeveer) 3 zinnen wat ze doen.

Wat is een library precies?

Een library is een set functies die het makkelijker maakt om bepaalde taken uit te voeren en kan in sommige gevallen zelfs de taak helemaal overnemen.


 1. Java.lang (Java)Fundamentele klassen en interfaces voor de kerntaal van Java. Hier vallen onder andere de Boolean, Byte, Float, Math en Process onder. Java is hier rechtstreeks van afhankelijk.
 2. Java.sql (Java)Door middel van deze library is het mogelijk om gegevens uit een SQL gegevensbron te halen.
 3. Algolia Places (JavaScript)Helpt bij het automatisch aanvullen van formulieren. Denk hierbij aan adressen. Verder kan er ook een kaart toegevoegd worden waar de locatie weergegeven word.
 4. Anime.js (JavaScript)Deze library maakt het mogelijk om animaties en bewegende objecten aan je website toe te voegen.
 5. AOS (Javascript)AOS maakt het mogelijk om ‘on scroll’ animaties toe te voegen aan je website. Zo krijg je een prachtig siteontwerp waarmee je je gebruikers betrokken houdt.
 6. Bideo.js (JavaScript)Deze library helpt je bij het invoegen van een video-achtergrond op je website. Kan worden aangepast naar elk schermformaat en schaalt makkelijk mee.
 7. Choreographer-js (JavaScript)Door het gebruik van deze library kunnen er complexe CSS-animaties op je site worden gemaakt. Een voorbeeld is te zien op hun website.
 8. Cleave.js (JavaScript)Cleave.js zorgt er voor dat content zoals een telefoonnummer direct word opgemaakt naar bijvoorbeeld “(+31) 06-15517712” zodat er alleen maar de getallen ingevoerd hoeven te worden door de gebruiker.
 9. Premonish (JavaScript)Premonish maakt het mogelijk om bewegingen van de muis te voorspellen. Dit kan zorgen voor hele interessante en coole animaties.
 10. ReactJS (JavaScript)Misschien wel de meest bekende JavaScript library uit dit hele rijtje. Maar onmisbaar om niet op te noemen, React. Met deze library kunnen er gebruikersinterfaces worden gebouwd.
